# 貼文格式模板（香港版）

各平台和內容類型的即用模板。

## 目錄
- LinkedIn 貼文模板（故事貼、反向觀點、列表貼、教學貼）
- Instagram / 小紅書模板（輪播貼、Reels 腳本）
- Facebook 模板（社群互動貼、活動推廣）
- 開場句公式（好奇心、故事、價值、反向、本地香港）
- 廣東話書面語寫作示例

## LinkedIn 貼文模板

### 故事貼（Story Post）
```
[Hook：意外成果或教訓]

[場景設定：什麼時候、在哪裏發生]

[你面對的挑戰]

[你嘗試了什麼 / 發生了什麼]

[轉捩點]

[結果]

[對讀者的啟示]

[引發互動的問題]
```

**香港故事貼示例：**
```
做咗 8 年 HR，今年才明白留人唔係靠人工。

一間本地初創，前年流失率超過 60%。
老闆第一個反應：加薪。

加了。走的人仍然走。

後來我們做了一件事，改變了整個局面：
我們問每一個員工：你喺呢度最大的阻礙係什麼？

答案令我們意外。
唔係人工，唔係福利。
係「感覺唔到自己嘅工作有意思」。

我們花了兩個月，幫每個人重新釐清工作目標。
流失率跌到 15%。

留住人才，要先留住他們的意義感。

你公司最大的員工挑戰係什麼？
```

### 反向觀點貼（Contrarian Take）
```
[大膽提出反向觀點]

原因係：

[原因 1]
[原因 2]
[原因 3]

[你建議的做法]

[邀請討論：「我係咪錯咗？」]
```

### 列表貼（List Post）
```
[X] 個我在 [背景] 學到關於 [話題] 的事：

1. [觀點] — [簡短說明]

2. [觀點] — [簡短說明]

3. [觀點] — [簡短說明]

[總結洞察]

你最有共鳴的係哪一點？
```

### 教學貼（How-To）
```
點樣 [達成成果]（[時間框架]內）：

第一步：[行動]
↳ [為什麼這一步重要]

第二步：[行動]
↳ [關鍵細節]

第三步：[行動]
↳ [常見錯誤要避免]

[預期結果]

[CTA 或問題]
```

## 小紅書 / Instagram 輪播貼模板

### 教育型輪播（10 頁）
```
第 1 頁：[吸睛標題 + 視覺]
「[X] 個 [類別] 技巧，[受眾] 一定要知道」

第 2-8 頁：[每頁一個要點]
- 大標題（一眼看懂）
- 1-2 句說明
- 視覺圖示 / 示例

第 9 頁：[總結]
「記住這 [X] 點：[簡列]」

第 10 頁：[CTA]
「儲存呢個貼文，下次用到唔使再搵！」
「追蹤我，每週更新 [類別] 內容」
```

### 小紅書筆記結構
```
標題（關鍵詞優先）：
「[地區/類別] [主題] [測評/推薦/教學] | [受眾]必看」

正文：
開場：[個人經歷或問題引入]

主體：
✨ [要點 1]：[說明]
💡 [要點 2]：[說明]
🔥 [要點 3]：[說明]

總結：[一句話核心建議]

結尾 CTA：
「覺得有用的話點個贊 ❤️」
「有問題留言問我喲！」

Hashtag：
#香港 #[主話題大類] #[細話題] #[品牌話題]
```

## Reels / YouTube Shorts 腳本模板

### 教學型短影片（30-60 秒）
```
開場（0-3 秒）：[衝擊性陳述或問題]
「你知唔知 [出人意料的事實]？」

鋪墊（3-8 秒）：[背景說明]
「好多人以為 [常見誤解]，但其實……」

核心內容（8-50 秒）：[逐步說明]
「第一：[要點]
 第二：[要點]
 第三：[要點]」

CTA（最後 5 秒）：[行動呼籲]
「覺得有用就 Follow，每週教你 [類別]！」
```

### 故事型短影片（30-60 秒）
```
開場（0-3 秒）：[最精彩的結果先說]
「我靠呢個方法，三個月多賺咗 [X]。」

故事（3-45 秒）：[壓縮版敘事]
「開始係咁的……[簡短過程]……然後我做了 [關鍵行動]……」

結果（45-55 秒）：[具體成果]
「結果係 [具體數字]。」

CTA（最後 5 秒）：
「想知細節？留言『想學』我告訴你！」
```

## 開場句公式（Hook Formulas）

Hook 的核心原則：**讓人停下來，讓人想繼續睇。**
情緒越強烈、越真實，效果越好。以下按情緒強度和使用場景分類。

---

### 😱 情緒衝擊 Hook（小紅書 / IG Reels 首選）

這類 Hook 靠強烈情緒和緊迫感抓住眼球，適合 B2C、生活風格、美容、飲食類內容。

**姊妹 / 兄弟呼喚式（建立親密感）**
- 「姊妹們聽我一句！[重要忠告]」
- 「姊妹們，我不是在開玩笑——[驚人事實]」
- 「各位，認真的，[緊急提醒]」
- 「朋友們，今次要嚴肅一下……[重要分享]」

**震驚揭露式（製造意外感）**
- 「太誇張了，我真的沒有想到[結果]。」
- 「我刷到一個[類別]，直接驚呆了。」
- 「不收藏你一定會後悔！[內容類別]大合集」
- 「看完之後我整個人都不好了……」
- 「這個[產品/方法/地方]真的太過分了（好的意思）」

**秘密揭露式（好奇心+緊迫感）**
- 「我剛發現了一個秘密，不分享出來良心過不去。」
- 「這個[行業]不會告訴你的事，我說了。」
- 「一個讓我[成果]的方法，很少人知道。」
- 「用了[X]年才搞清楚，現在告訴你免你走彎路。」

**懊悔時光式（引發共鳴）**
- 「如果時間能重來，我一定會先知道[這件事]。」
- 「後悔沒有早點發現[產品/方法/地方]。」
- 「早知道就好了！[主題] 避雷指南。」
- 「當年要是有人告訴我[建議]，我能少走多少彎路。」

**完蛋警報式（自嘲+共鳴）**
- 「完蛋了，我又[忍不住做了某事]。」
- 「麻煩大了，[問題]，但我找到解決方法了。」
- 「我真的救不了我自己，[情景描述]。」
- 「老實說，我有點上癮了……[誠實分享]」

---

### 🤔 好奇心 Hook

適合教育類、知識類、B2B 內容。

- 「我一直以為 [常見觀念]，直到發生了這件事。」
- 「[令人意外的結果]——而且只花了 [出乎意料的短時間]。」
- 「原來 [成果] 背後，真正的原因係……」
- 「冇人講過你，[行業] 入面最大的秘密係……」
- 「為什麼 [常見現象]？答案比你想的簡單。」

---

### 📖 故事 Hook

適合個人品牌、創業故事、真實經歷分享。

- 「上個星期，[意想不到的事] 發生了。」
- 「我差點 [重大錯誤 / 失敗]。」
- 「[X] 年前，我 [過去狀態]。今天，[現狀]。」
- 「有個客戶問咗我一個問題，我突然唔識答。」
- 「那一刻我才明白，[深刻體會]。」

---

### 💡 價值 Hook

適合教學貼、攻略、工具分享。

- 「點樣 [達成理想成果]（而唔需要 [常見障礙]）：」
- 「[數字] 個 [事物]，幫你 [成果]：」
- 「唔好再 [常見錯誤]。改做呢個：」
- 「最簡單嘅方式去 [成果]，只需要 [步驟數] 步：」
- 「不收藏你一定後悔的 [類別] 清單 ✅」

---

### 🔥 反向觀點 Hook

適合引發討論、建立思想領導力。

- 「我唔同意大家話的 [常見建議]，原因係……」
- 「[常見建議] 係錯的。點解：」
- 「我停止了 [常見做法]，然後 [正面結果]。」
- 「大家都話要 [X]，但係……」

---

### 🇭🇰 香港本地 Hook

適合有本地身份認同的內容，引起在地共鳴。

- 「做咗 [X] 年 [行業]，終於明白……」
- 「係香港 [做某事]，最難的唔係 [常見答案]，係……」
- 「好多人問我點解 [成果]，答案其實好簡單：」
- 「係呢座城市打工 / 創業，你一定有過呢種感覺：」
- 「香港人的[類別]攻略，外面真的找不到。」

---

### 👥 社會認同 Hook

適合有成果數據、客戶案例可展示時。

- 「我哋用 [X 個月] 做到了 [結果]。完整故事：」
- 「[數字] 個人問過我關於 [話題]。以下係我的答案：」
- 「[數字] 個人收藏了這篇，你還沒看？」
- 「我跟了 [某人 / 方法] 學了 [教訓]。」

---

### Hook 使用指引

| 平台 | 最適合的 Hook 類型 |
|------|-----------------|
| 小紅書 | 情緒衝擊、懊悔時光、秘密揭露、價值（不收藏後悔） |
| Instagram Reels | 情緒衝擊、姊妹呼喚、完蛋警報、故事 |
| LinkedIn | 好奇心、故事、反向觀點、香港本地、社會認同 |
| Facebook | 故事、香港本地、社會認同、好奇心 |
| YouTube Shorts | 震驚揭露、秘密揭露、價值、故事 |

**⚠️ 注意：** 情緒衝擊 Hook 效果強，但要確保內容貨真價實——過度使用或言過其實會損害長期信任度。每 3-4 篇用一次，搭配其他類型的 Hook 輪換使用。

## 廣東話書面語寫作示例

### 同樣內容的三種語言風格

**正式中文書面語（適合企業 / 政府）：**
「本公司致力於為客戶提供最優質的服務體驗，並持續優化業務流程以達成卓越績效。」

**廣東話書面語（適合本地品牌 / 個人）：**
「我哋一直努力做到最好，唔斷改善，希望你用得開心。」

**中英夾雜（適合 B2B / 科技）：**
「我們 team 花咗 3 個月 develop 呢個 solution，終於上線喇！」

### 廣東話常用轉換

| 普通話書面語 | 廣東話書面語 |
|------------|------------|
| 是 | 係 |
| 不是 / 沒有 | 唔係 / 冇 |
| 為什麼 | 點解 |
| 怎麼 / 如何 | 點樣 / 點 |
| 這個 / 那個 | 呢個 / 嗰個 |
| 很 / 非常 | 好 / 幾 |
| 告訴 | 話俾知 |
| 試一試 | 試吓 |
| 知道了 | 知喇 |
| 可以 | 得㗎 / 可以 |
