## 登陸頁版塊類型

### 核心版塊

**英雄區（Above the Fold）**
- 標題 + 副標題
- 主要 CTA
- 配套視覺（產品截圖、英雄圖片）
- 可選：社會認同條

**社會認同條**
- 客戶標誌（知名度 > 數量）
- 關鍵數字（「超過 500 間香港企業使用」）
- 星級評分與評論數量
- 短評語片段

**問題／痛點版塊**
- 比客戶更準確地描述他們的問題
- 製造認同感（「係呀，我就係咁」）
- 暗示不解決問題的代價

**方案／利益版塊**
- 從問題過渡到你的方案
- 3-5 個核心利益（唔係 10 個）
- 每項：標題 + 說明 + 佐證（如有）

**操作流程**
- 3-4 個編號步驟
- 降低感知複雜度
- 每步：行動 + 成果

**最終 CTA 版塊**
- 重申價值主張
- 重複主要 CTA
- 風險逆轉（保證、免費試用）

---

### 支援版塊

**客戶評語** — 具體成果 > 模糊讚美；附姓名、職位、公司

**案例研究** — 問題 → 方案 → 成果；具體數字

**使用場景** — 幫助訪客自我代入；「適合需要 X 的人士」格式

**目標受眾版塊** — 「最適合[職位]」格式；解答「這個是否適合我？」

**FAQ 版塊** — 解答常見疑慮；5-10 個最常見問題

**競品對比版塊** — vs. 競爭對手或 vs. 現有做法（Excel、手工流程）

**整合／合作伙伴** — 配合使用工具的標誌；「配合你現有工具」訊息

**創辦人故事** — 為什麼建立這個產品；情感連結

**示範／產品導覽** — 互動示範、影片、GIF

**保證／風險逆轉** — 退款保證、免費試用條款、「隨時取消」

**數據版塊** — 建立可信度的關鍵數字（客戶數、評分、節省金額）

## 頁面結構模板

### 功能堆砌頁（弱）

```
1. 英雄區
2. 功能一
3. 功能二
4. 功能三
5. CTA
```

這是一份清單，不是說服敘事。

---

### 說服敘事頁（強）

```
1. 英雄區（清晰價值主張）
2. 社會認同條（標誌或數字）
3. 問題／痛點版塊
4. 操作流程（3 個步驟）
5. 核心利益（2-3 項，唔係 10 項）
6. 客戶評語
7. 使用場景或目標受眾版塊
8. 與替代方案對比
9. 案例研究摘要
10. FAQ
11. 最終 CTA（附保證）
```

---

### 緊湊登陸頁

```
1. 英雄區（標題、副標題、CTA、圖片）
2. 社會認同條
3. 3 個核心利益（附圖示）
4. 客戶評語
5. 操作流程（3 個步驟）
6. 最終 CTA（附保證）
```

適用於廣告登陸頁。

---

### 企業 B2B 登陸頁

```
1. 英雄區（成果導向標題）
2. 標誌條（知名企業客戶）
3. 問題版塊（業務層面痛點）
4. 方案概覽
5. 按職位／部門的使用場景
6. 安全性／合規版塊
7. 整合標誌
8. 附數據的案例研究
9. 投資回報率版塊
10. 聯絡／示範 CTA
```

---

### 香港本地服務頁（新增）

```
1. 英雄區（點名服務地區：「服務全港」）
2. 本地客戶評語（附公司名稱）
3. 常見本地痛點版塊
4. 服務內容說明
5. 操作流程（3-4 步）
6. 資質認證或業界認可
7. FAQ（包括價格透明度）
8. 多渠道 CTA（網頁表單 + WhatsApp + 電話）
```

適用於本地 SME 服務、專業服務、零售業。

---

### 產品發布頁

```
1. 英雄區（附發布公告）
2. 影片示範或產品導覽
3. 功能亮點（3-5 項）
4. 前後對比
5. 早期用戶評語
6. 發布優惠或搶先體驗
7. CTA（附緊迫感）
```

## 各版塊寫法提示

### 問題版塊開頭語
- 「你有沒有試過……」
- 「如果你係[職位]，你一定明白……」
- 「每天，[受眾群體]都要面對……」
- 「我們都試過……」

### 利益版塊結構
每項利益包含：標題（成果）+ 內文（如何實現，1-2 句）+ 佐證（數字或例子，可選）

### 操作流程步驟結構
- 編號：製造進度感
- 簡單動詞：「連接」、「設定」、「獲取」
- 成果導向：說明這步驟帶來什麼

例：
1. 連接你的工具（只需 2 分鐘）
2. 設定你的偏好
3. 每個星期一收到自動報告

### 客戶評語篩選

最好的評語包含：
- 具體成果（「節省了 32% 的處理時間」）
- 前後對比（「以前我們要花幾個小時……」）
- 職位 + 公司名稱

避免使用：「非常好用！」、「強烈推薦！」、「很容易上手！」

### 香港特有的社會認同形式
- **本地知名企業客戶標誌**（比外國品牌更有說服力）
- **業界獎項或認證**（如 FinTech Award、香港生產力局認可等）
- **媒體報道**（《信報》、《經濟日報》、《明報》等）
- **用戶數量**（「超過 X 間香港企業選用」比泛泛的全球數字更有感）
- **WhatsApp 截圖評語**（本地消費者更信任，顯得真實）
