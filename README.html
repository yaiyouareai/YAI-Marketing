快速安裝
一行指令，即刻完成

將技能套件上傳至 GitHub 後，用戶可以直接用以下指令安裝，無需手動下載任何檔案。

前置要求：已安裝 OpenClaw。技能會安裝至 ~/.openclaw/skills/ 目錄。
1

一行指令安裝（推薦）

打開終端機（macOS：Terminal / iTerm；Windows：PowerShell 或 WSL），貼上以下指令：

curl -fsSL https://raw.githubusercontent.com/yaiyouareai/YAI-Marketing/main/install.sh | bash

腳本會自動下載所有檔案並安裝至 OpenClaw 的技能目錄。

Terminal
$ curl -fsSL https://raw.githubusercontent.com/yaiyouareai/YAI-Marketing/main/install.sh | bash

Claude.ai 透過 Projects 功能支援技能——將技能的 SKILL.md 內容加入 Project Instructions，或上傳檔案至 Project Knowledge。以下步驟針對 Claude.ai Pro / Team 用戶。
1

建立新 Project

在 Claude.ai 左側欄點擊 Projects，然後點擊 + New Project，命名為「香港市場行銷」或任何你喜歡的名稱。

2

上傳技能檔案至 Project Knowledge

在 Project 設定頁面，找到 Project Knowledge 區域，點擊 Add content，逐一上傳以下檔案：

  • product-marketing-context-hk/SKILL.md
  • copywriting-hk/SKILL.md
  • copywriting-hk/references/copy-frameworks-hk.md
  • copywriting-hk/references/natural-transitions-hk.md
建議：上傳時保留原始檔名，方便日後識別和更新。
3

設定 Project Instructions

在 Project 設定中找到 Custom Instructions 或 Project Instructions，加入以下說明：

# 技能指引

你有兩個香港市場行銷技能，已上傳至 Project Knowledge：

1. product-marketing-context-hk（SKILL.md）
   - 建立和管理產品行銷背景資料
   - 儲存於 .agents/product-marketing-context.md

2. copywriting-hk（SKILL.md）
   - 撰寫香港市場推廣文案
   - 需要時參考 copy-frameworks-hk.md 和
     natural-transitions-hk.md

處理行銷相關任務時，先讀取對應的 SKILL.md
並按照其中的指引執行。
4

（可選）預載產品背景資料

如果你已有產品背景資料，可以將 .agents/product-marketing-context.md 也上傳至 Project Knowledge。這樣每次對話都能自動引用，唔需要重新輸入。

Claude.ai — Project Knowledge
Project Knowledge
✓ SKILL.md (product-marketing-context-hk)
✓ SKILL.md (copywriting-hk)
✓ copy-frameworks-hk.md
✓ natural-transitions-hk.md
✓ product-marketing-context.md ← 可選
5

在 Project 內開始對話

確保在你建立的 Project 內開啟新對話（唔係普通的 New Chat），然後直接輸入觸發語句：

Claude.ai — 香港市場行銷 Project
你 →
幫我寫主頁文案，我係做 HR 軟件的
── Claude ─────────────────────────────
我先查看一下你的產品行銷背景資料……
[讀取 copywriting-hk SKILL.md]
找到了。根據你的產品背景，以下是主頁文案建議……
重要：必須在 Project 內對話，普通的 New Chat 無法讀取 Project Knowledge 和 Instructions。
