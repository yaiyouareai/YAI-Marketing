先安裝
①
product-marketing-context-hk

產品行銷背景資料

建立一份產品定位文件，記錄受眾、競爭格局、客戶語言。其他五個技能自動讀取，唔需要重複輸入。

②
strategy-content-hk

內容策略

規劃話題群組、關鍵詞研究、內容日曆，包括香港本地渠道（連登、Facebook 群組）分析。

③
strategy-launch-hk

產品發布策略

規劃發布 ORB 框架、香港媒體策略、KOL 合作，以及本地社群（Cyberport、科學園）渠道。

④
copywriting-hk

文案撰寫

撰寫主頁、登陸頁、定價頁等各類推廣文案，符合香港語境，支援廣東話書面語和中英夾雜。

⑤
editing-content-hk

文案編輯

用七步編輯框架改善現有文案，專門識別翻譯腔、企業官腔和香港特有的語言問題。

⑥
social-content-hk

社交媒體內容

規劃 LinkedIn、Instagram、小紅書、Facebook、WhatsApp 廣播的內容計劃，含香港 Hook 公式庫。

快速安裝
選擇你的平台

根據你使用的工具，按照對應的步驟安裝。

前置要求：已安裝 OpenClaw。技能會安裝至 ~/.openclaw/skills/。
1

一行指令安裝全部（推薦）

打開終端機，貼上以下指令：

curl -fsSL https://raw.githubusercontent.com/yaiyouareai/YAI-Marketing/main/install.sh | bash
Terminal
$ curl -fsSL .../install.sh | bash

2

只安裝指定技能（可選）

# 只安裝文案技能
... | bash -s -- --skill copywriting-hk

# 只安裝社交媒體技能
... | bash -s -- --skill social-content-hk

# 指定安裝目錄
... | bash -s -- --dir ~/my-skills

可用技能名稱：product-marketing-context-hk copywriting-hk editing-content-hk social-content-hk strategy-content-hk strategy-launch-hk

3

在 OpenClaw 確認安裝成功

打開 OpenClaw → 左側欄 Skills，確認六個技能已列出：

OpenClaw — Skills
✓ product-marketing-context-hk v1.0.0
✓ strategy-content-hk v1.0.0
✓ strategy-launch-hk v1.0.0
✓ copywriting-hk v1.0.0
✓ editing-content-hk v1.0.0
✓ social-content-hk v1.1.0
如技能未顯示，嘗試重啟 OpenClaw 或手動刷新 Skills 頁面。
4

開始使用

在 OpenClaw 開啟新對話，輸入觸發語句：

OpenClaw — New Chat
你 →
幫我設定產品行銷背景資料
── Claude ─────────────────────────
Using skill: product-marketing-context-hk
好的，我先檢查是否已有背景資料……
適用於：Claude.ai Pro / Team 用戶。透過 Project 功能手動使用技能。
注意：必須在 Project 內對話——普通的 New Chat 無法讀取技能檔案。
1

建立新 Project

在 Claude.ai 左側欄點擊 Projects → + New Project，命名為「YAI 香港行銷技能」。

2

上傳 SKILL.md 檔案至 Project Knowledge

在 Project 設定找到 Project Knowledge → Add content，上傳以下核心檔案：

  • product-marketing-context-hk/SKILL.md
  • strategy-content-hk/SKILL.md
  • strategy-launch-hk/SKILL.md
  • copywriting-hk/SKILL.md
  • editing-content-hk/SKILL.md
  • social-content-hk/SKILL.md
3

上傳 Reference Files（建議）

一併上傳以下參考檔案，讓技能回應更完整：

  • copywriting-hk/references/copy-frameworks-hk.md
  • copywriting-hk/references/natural-transitions-hk.md
  • editing-content-hk/references/plain-english-alternatives-hk.md
  • social-content-hk/references/platforms-hk.md
  • social-content-hk/references/post-templates-hk.md
  • social-content-hk/references/reverse-engineering-hk.md
  • strategy-content-hk/references/headless-cms-hk.md
4

設定 Project Instructions

在 Project 設定找到 Custom Instructions，加入以下說明：

你有六個香港市場行銷技能，已上傳至 Project Knowledge：

1. product-marketing-context-hk — 建立和管理產品行銷背景資料
2. strategy-content-hk — 規劃內容策略和話題群組
3. strategy-launch-hk — 規劃產品發布策略
4. copywriting-hk — 撰寫香港市場推廣文案
5. editing-content-hk — 編輯和改善現有文案
6. social-content-hk — 建立社交媒體內容計劃

處理行銷任務時，先讀取對應的 SKILL.md 並按照其中的
指引執行。背景資料儲存於 .agents/product-marketing-context.md。
5

在 Project 內開啟對話

確保在你建立的 Project 內開啟新對話，然後輸入觸發語句：

Claude.ai — YAI 香港行銷技能 Project
你 →
幫我寫主頁文案，我係做 HR 軟件的
── Claude ──────────────────────────
我先查看你的產品行銷背景資料……
[讀取 copywriting-hk SKILL.md]
