# 簡明用語替換表（香港版）

用更簡單、更自然的替代品取代複雜或企業腔的詞語。

來源：Plain English Campaign（2001）、澳洲政府文體手冊（2024）、plainlanguage.gov，以及香港中文使用場景補充。

## 英文企業術語替換

常見於中英夾雜文案中的弱英文詞，建議替換：

| 企業英文 | 更好的替換 |
|---------|----------|
| Utilize | Use |
| Implement | Set up / Do |
| Leverage | Use |
| Facilitate | Help |
| Innovative | New |
| Robust | Strong |
| Seamless | Smooth |
| Cutting-edge | New / Modern |
| Comprehensive | Complete / Full |
| Streamline | Simplify |
| Optimize | Improve |
| Synergy | Working together |
| Value-add | Added value / Benefit |
| Pain point | Problem |
| Solution | （直接說解決什麼） |
| Empower | Let / Help |
| Game-changer | （直接說有什麼不同） |

## 香港中文翻譯腔替換

這些詞語通常從英文逐字翻譯，讀落不自然：

| 翻譯腔 / 企業腔 | 自然中文替換 |
|---------------|------------|
| 致力為您提供 | 幫你 / 為你 |
| 以達致卓越績效 | 做得更好 / 取得更好成果 |
| 就您的需求而言 | 對你來說 / 對你的情況 |
| 業界翹楚 | 行業領先（要有佐證） |
| 一站式解決方案 | （直接說解決什麼問題） |
| 全方位服務 | （直接說提供什麼） |
| 賦能 | 讓……能夠 / 幫助 |
| 精準 | 準確 |
| 無縫銜接 | 流暢配合 / 輕鬆整合 |
| 深度賦能 | 真正幫到 |
| 全面升級 | 大幅改善 |
| 深度整合 | 緊密連接 |
| 協同效應 | 合作效果 |
| 閉環管理 | 完整流程 |
| 頂層設計 | 整體規劃 |
| 構建生態 | 建立體系 |

## 廣東話書面語替換

常見於香港文案的冗長表達，可以更簡短：

| 冗長表達 | 更簡短的表達 |
|---------|------------|
| 是 | 係（廣東話書面語） |
| 不是 / 沒有 | 唔係 / 冇 |
| 為什麼 | 點解 |
| 怎麼 / 如何 | 點樣 / 點 |
| 因此 | 所以 |
| 但是 | 但係 |
| 非常 / 很 | 好（例：好有用） |
| 現在 | 而家 |
| 已經 | 已經（廣東話相同） |
| 告訴 | 話俾知 |
| 試一試 | 試吓 |
| 這樣做 | 咁做 |
| 可以的 | 得㗎 |
| 馬上 | 即刻 |
| 繼續 | 繼續 / keep 住 |

## 完全刪除的短語

這些短語通常什麼都沒有說。直接刪除：

**英文填充詞：**
- A total of（直接說數字）
- Absolutely
- Actually
- At the end of the day
- At this moment in time
- Basically
- Currently（如果「現在」或什麼都不說更好）
- Obviously
- Of course
- Quite
- Really
- Very

**中文填充詞：**
- 在此基礎上
- 值得注意的是
- 不言而喻
- 眾所周知
- 毋庸置疑
- 可以說
- 我認為（直接說你的觀點）
- 在某種程度上
- 從某種意義上說
- 總的來說（在結尾沒說新東西時）

**香港中文特有填充詞：**
- 「其實」（用過多時）
- 「其實係」（拖慢節奏）
- 「所謂的」（常常不必要）
- 「可以話係」（直接說是）
- 「某程度上嚟講」（說清楚程度，或直接說）
